Que es la memoria

la memoria es un componente vital en la arquitectura de computadoras y dispositivos electrónicos, ya que permite el almacenamiento temporal de datos para un acceso rápido, facilitando el funcionamiento eficiente del sistema.

Tipo de Memoria Descripción imagen
RAM Memoria de acceso aleatorio, volátil y rápida La memoria de acceso aleatorio (RAM) es la memoria de la computadora que almacena la información que un programa necesita mientras se ejecuta.
ROM de solo lectura, no volátil, usada para el firmware La memoria ROM (Read Only Memory) es el almacenamiento que ofrece un dispositivo a través de una tecnología NAND Flash, como puede ser eMMC, UFS o NVMe. En la ROM se almacenan datos e instrucciones y está integrada dentro del SoC del dispositivo, ya sea una Tablet, smartphone, portátil o, incluso, una Smart TV.
Cache Memoria rápida que almacena datos temporales En informática, una memoria caché es una capa de almacenamiento de datos de alta velocidad que almacena un subconjunto de datos, normalmente transitorios, de modo que las solicitudes futuras de dichos datos se atienden con mayor rapidez que si se debe acceder a los datos desde la ubicación de almacenamiento principal.
Virtual Memoria que simula RAM usando espacio en disco La memoria virtual es una técnica que utilizan los ordenadores para trasladar datos de la memoria de acceso aleatorio (RAM) al disco duro cuando la cantidad de RAM es limitada.
